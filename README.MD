# İstinye Uni Case

Bu proje, **İstinye Üniversitesi** için geliştirilmiş bir **Case Study** projesidir. Frontend tarafı **Angular 18**, **TailwindCSS** ve **Angular Material** ile yazılmıştır, backend tarafı ise **Express.js** kullanılarak oluşturulmuştur.


## Proje Kurulumu ve Çalıştırma

### 1. Backend Kurulumu ve Çalıştırılması

Backend, **Express.js** kullanılarak geliştirilmiştir. Aşağıdaki adımları izleyerek backend tarafını çalıştırabilirsiniz.

#### Adım 1: Backend klasörüne gidin

```bash
cd backend
```

#### Adım 2: Gerekli bağımlılıkları yükleyin

```bash
npm install
```

#### Adım 3: Backend'i çalıştırın

```bash
node index.js
```

Express.js backend uygulamanız şu anda çalışıyor olacak ve API'niz kullanıma hazır olacaktır. Varsayılan olarak, backend genellikle **http://localhost:3000** adresinde çalışır. API uç noktalarını kullanmaya başlayabilirsiniz.

### 2. Frontend Kurulumu ve Çalıştırılması

Frontend, **Angular 18** ile geliştirilmiştir. Angular CLI'yi kullanarak frontend'i çalıştırabilirsiniz.

#### Adım 1: Frontend klasörüne gidin

```bash
cd frontend
```

#### Adım 2: Gerekli bağımlılıkları yükleyin

```bash
npm install
```

#### Adım 3: Angular uygulamasını başlatın

```bash
ng serve --open
```

Bu komut, frontend uygulamanızı **http://localhost:4200** adresinde çalıştıracaktır. Tarayıcınızda bu adresi ziyaret ederek uygulamanızı görüntüleyebilirsiniz.
